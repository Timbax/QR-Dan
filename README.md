# QR Generator

Generador de códigos QR moderno construido con React y Vite.

## Tecnologias

- **React 19** - Libreria UI
- **Vite 7** - Build tool
- **Bun** - Package manager
- **ESLint** - Linting

## Instalacion

```bash
bun install
```

## Como ejecutar

### Desarrollo

```bash
bun run dev
```

### Produccion

```bash
bun run build
bun run preview
```

### Linting

```bash
bun run lint
```
